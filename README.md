# Яхёев Абдурахим Фирузович

## Задача № 5 / Алгоритмы

## Чтобы запустить программу у вас должен быть установлен компилятор для языка С++ : gcc (Ubuntu 13.3.0-6ubuntu2~24.04) 13.3.0 , и выполните следующую команду для компиляции и запуска: g++ main.cpp -o main; ./main

## Алгоритм решения:

Минотавр начинает в стартовой клетке с заданным направлением, эту клетку сразу помечаем посещённой. Далее проверяем выгодно ли развести костёр (C < 4A), если так делаем это действие, чтобы узнать ближайшие клетки. Далее у нас будет цикл обработки, пока есть не посещённые клетки или не проверены все граничные клетки, делаем следующее :

- если всё посетили и всё проверили, то всё, задача решена
- смотрим на известные, но не посещённые клетки.Проверяем направлени из известных клеток, где ещё не знаем, стена там или новая клетка. Используем алгоритм Дейкстры, чтобы найти кратчайший путь по времени до каждой цели. Выбираем клетку с минимальным затратом времени, в приоритете посещения клетки перед проверкой возможности двигаться в направлении. Выполняем шаги и повороты для этого пути. Когда добрались до клетки, разводим костёр если выгодны, иначе пробуем сделать шаг, то есть пробу, если есть клетка, отмечаем посещённой иначе стенаи отмечаем это направление как проверенное. После каждого действия обновляем информацию про наши клетки(у нас по сути связный граф получается).
  Когда посетили и проверили все клетки, выводим общее потраченное время.
